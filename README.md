# Listmonk

Email newsletter and mailing list manager powered by Listmonk

## Deployment

- **Hostname**: mail.megabyte.space
- **Image**: `knadh/listmonk:latest`
- **Port**: 9000
- **Infrastructure**: Cloudflare Workers + Containers
- **Database**: Neon Postgres
